O nama

Ovaj web shop izrađen je kao školski projekt na kojem smo radili u paru. Imamo 16 godina i dolazimo iz Imotskog, a cilj nam je bio napraviti modernu i funkcionalnu online trgovinu. Tijekom izrade projekta koristili smo HTML, CSS (Tailwind) i JavaScript kako bismo omogućili pregled proizvoda, dodavanje u košaricu i jednostavan proces narudžbe. Kroz ovaj rad naučili smo kako izraditi responzivnu web stranicu, organizirati proizvode po kategorijama te poboljšati korisničko iskustvo. Projekt nam je pomogao razviti vještine timskog rada, rješavanja problema i kreativnog razmišljanja, a ujedno smo stekli i nova znanja iz web razvoja. Hvala vam na posjeti našem web shopu!
